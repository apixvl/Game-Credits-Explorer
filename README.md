# Retro Game Credits Exploration
Right now this is designed to be an simple SPA exploration of many different obscure and old video games' credits, to see who was behind these incredible games. It is SEVERELY work in progress. Do not expect updates very often, this is just a passion project.

While the site is not open to suggestions on games to add or site features, I would like it if you could find missing credits inside of games already added. there are quite a few games where I cannot find credits for specific versions, or if there are any missing consoles the game has appeared on or missing persons that were not credited.
